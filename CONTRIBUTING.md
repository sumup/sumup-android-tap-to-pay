# Contributing

Thank you for your interest in this repository. This is a sample app for the SumUp Android TapToPay SDK.

## Contribution Policy

This repository is **not open for external contributions**.

We do not accept:
- Pull requests
- Issues
- Feature requests
- Bug reports from external contributors

## Why

This repository is maintained internally and is published for transparency and reference purposes only. All changes are managed by the internal team following strict security and compliance processes.

## Security

If you discover a security vulnerability, please follow the instructions in our [SECURITY.md](./SECURITY.md) file.

## Ownership

All changes are handled by the designated code owners.
## Support

For usage and integration guidance, refer to the documentation linked in the README.
